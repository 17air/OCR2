# OCR + NER Android Sample

This repository contains a simple Android application demonstrating how to perform
OCR on a business card image and run NER (named entity recognition) on the
extracted text. The project is written in Kotlin.

The NER model file (`ner_model_fp16.tflite`) should be placed under
`app/src/main/assets/` after cloning the repository.

This project does not depend on any previous code base and can be imported as a
stand‑alone Android Studio project.
